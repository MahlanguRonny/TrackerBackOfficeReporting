# Trader BackOffice Reporting App

# Follow guidllines below to run the project/system

Clone the repository
  https://github.com/MahlanguRonny/TrackerBackOfficeReporting
  
  1. Open the project in visual studio and run nuget restore
  
  2. Set multiple start up projects on visual studio(both wcf and core api)
  
  3. Click start and if all is well both apps should open and ready for testing
  4. **Very important** check the wcf service url in the app setting json file, make sure it's the as the one that open in your browser, if not please change app settings value
